# UI Parity Handbook

Dokumen ini menjadi jalur review resmi untuk memastikan surface Bidan aktif kembali setara dengan `bidanapp copy`.

## Cara pakai

1. Buka screen aktif di `bidan.lvh.me`.
2. Bandingkan dengan source of truth di `[/Users/adi/Code/startup/bidanapp copy](/Users/adi/Code/startup/bidanapp%20copy)`.
3. Cek empat lapisan yang wajib:
   - layout shell dan spacing rhythm
   - recipe card/input/chip/button
   - section composition dan hierarchy
   - flow/state behavior
4. Validasi hasil akhir dengan:
   - ledger di `matrix.md`
   - latest Playwright report
   - screen checklist di folder `screens`
